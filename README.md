# TFG - Credit Scoring con modelos lineales generalizados

Este repositorio contiene el código y los notebooks utilizados en mi Trabajo de Fin de Grado sobre credit scoring con modelos lineales generalizados, incluyendo modelos logit, probit y técnicas de tratamiento del desbalanceo como SMOTE.

## Contenido

* `notebooks/`: notebooks de Google Colab con el análisis
* `src/`: scripts auxiliares
* `figures/`: figuras generadas para la memoria
* `results/`: resultados y tablas exportadas

## Tecnologías utilizadas

* Python
* pandas
* numpy
* scikit-learn
* statsmodels
* matplotlib
* imbalanced-learn

## Objetivo

Analizar la probabilidad de impago en un contexto de riesgo crediticio y comparar el rendimiento de modelos econométricos tradicionales frente a enfoques de machine learning.
